1 - Instalar las fuentes:

ttf-ancient-fonts (For Linux)

sudo apt -y install ttf-ancient-fonts

Source Code Pro for Powerline (For windows terminal)

https://github.com/powerline/fonts/blob/master/SourceCodePro/Source%20Code%20Pro%20for%20Powerline.otf

Fira Mono for powerline (For Mac)

brew tap homebrew/cask-fonts
brew install --cask font-fira-mono-for-powerline

2 - Instalar Bullet-train:

wget -O $ZSH_CUSTOM/themes/bullet-train.zsh-theme http://raw.github.com/caiogondim/bullet-train-oh-my-zsh-theme/master/bullet-train.zsh-theme
